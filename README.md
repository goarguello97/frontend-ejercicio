# Frontend para Ejercicio Fullstack

Este repositorio contiene la parte visual de un proyecto desarrollado con React. Está diseñado para crear una interfaz de usuario moderna y dinámica utilizando varias bibliotecas populares.

## Descripción

Este proyecto utiliza React y varias herramientas del ecosistema de React, incluyendo Redux para la gestión del estado, React Router para la navegación y Bootstrap para el diseño responsivo.

## Instalación

Para instalar las dependencias del proyecto, ejecuta:

```bash
  npm install
```

Asegúrate de tener un archivo .env configurado en la raíz del proyecto para las variables de entorno necesarias.

## Variables de Entorno

Antes de ejecutar el proyecto, completa las siguientes variables de entorno en un archivo .env en la raíz del proyecto:

```bash
REACT_APP_API_URL=url_de_backend
```

## Uso

Para iniciar la aplicación, utiliza el siguiente comando:

```bash
  npm start

```

## Scripts

### start:

Inicia la aplicación en modo de desarrollo.

## Dependencias

Este proyecto utiliza las siguientes dependencias:

- axios: Cliente HTTP para hacer peticiones.
- bootstrap: Framework CSS para diseño responsivo.
- react: Biblioteca principal para construir interfaces de usuario.
- react-bootstrap: Componentes de Bootstrap para React.
- react-redux: Integración de Redux con React.
- react-router-dom: Biblioteca para manejar rutas en aplicaciones React.
